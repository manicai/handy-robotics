# Handy Robotics

A list of handy links for a robot builder. In time it may become [Awesome](https://github.com/sindresorhus/awesome) but my focus is on stuff I find useful not on being encyclopedic.

## Raspberry Pi

- [RaspberryPi.org](https://www.raspberrypi.org/): the Mothership.
- [Explorer pHat](https://shop.pimoroni.com/products/explorer-phat): interface board for the Pi Zero and [its software library](https://github.com/pimoroni/explorer-hat)
- [PiCon Zero](https://4tronix.co.uk/blog/?p=1224): "Intelligent Robotic Controller for Raspberry Pi"
- [RedBoard+](https://redrobotics.co.uk/): robot controller board.

## Shops (UK Focused)

- [4Tronix](https://shop.4tronix.co.uk/)
